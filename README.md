# VisibleChooser
VisibleChooser is many views visible manager library for Android.

## How do I use it?

### Setup
```
app gradle file

dependencies {
    compile 'com.github.paraya3636:VisibleChooser:0.1'
}
repositories {
    maven { url 'https://jitpack.io' }
}
```

# SDK Version
Build  
minSdkVersion:15 Android4.0.3+
